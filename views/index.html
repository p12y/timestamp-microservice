Timestamp Microservice

How to Use

  1. Pass a string as a parameter, and the API with check to see if it contains either a unix timestamp or a natural language date in any format compliant with RFC-1123 or ISO-8601 (example: January 1, 2016).
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Example Output
